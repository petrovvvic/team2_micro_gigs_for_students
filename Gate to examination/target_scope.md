# High-level Überblick

# Seitenbeschreibungen
Wireframes finden Sie im Unterordner "wireframes_imgs" gespeichert als PDF Deck

**Landing:**
Headline, kurze Erklärung, CTA-Button → Rechner

**Rechner:**
Eingabefelder (Fläche, Energieträger, Verbrauch, PLZ), Ergebnis: Ersparnis/Jahr + Amortisationszeit

**Angebotsvergleich:**
Liste von Angeboten (Name, Typ, COP, Preis), Filter, "Bestes Angebot" Badge

**Meine Daten:**
Formular: Name, Adresse, Fläche, gespeicherte Vergleiche anzeigen

**Wiki:**
Liste von Artikeln (Titel + Kurzbeschreibung), anklickbar → Detailseite
